Fix Email Links Domain + Add Send Attendee Email Action

Two issues need to be addressed:

All system-generated email links must use the production domain

The Attendees table must include a Send Scheduling Email action

These were partially implemented but are not working correctly.

1. Fix All Email Hyperlinks to Use the Production Domain

Currently emails contain links such as:

https://converge-concierge.replit.app/event/TLS2026

This must be replaced with the production domain.

Required behavior

All email hyperlinks generated by the system must point to:

https://concierge.convergeevents.com

Example corrected link:

https://concierge.convergeevents.com/event/TLS2026
2. Replace Hardcoded Domain References

Audit the codebase for any hardcoded references to:

converge-concierge.replit.app

Replace them with a configurable base URL.

Recommended approach:

Create a BASE_APP_URL environment variable.

Example:

BASE_APP_URL=https://concierge.convergeevents.com

All system-generated links should be constructed using this variable.

3. Update Email Templates

Check all email templates and update links accordingly.

Examples include:

Meeting scheduling emails
/event/{eventCode}
Sponsor report links
Dashboard links
Attendee scheduling links
Info request links
Deliverable notifications

All should build URLs using:

BASE_APP_URL + path
4. Verify Email Link Types

Ensure these links point to the correct domain:

Email Type	Expected URL
Attendee Scheduling Email	/event/{eventCode}
Sponsor Dashboard Access	/sponsor-dashboard
Sponsor Report Links	/reports/{sponsor}
Meeting Scheduling	/event/{eventCode}

All must use:

https://concierge.convergeevents.com
5. Add "Send Scheduling Email" Button to Attendees Table

The Attendees section currently shows only:

View

Edit

The Send Scheduling Email action is missing.

This must be added.

6. Attendees Table Row Actions

Update the attendee table row actions to include:

View

Generate a secure attendee-specific scheduling link

Send the attendee an email with the scheduling button

Use the correct event code

https://concierge.convergeevents.com/event/FRC2026

This fix is complete when:

• all system-generated email links use https://concierge.convergeevents.com
