========================================
SCHEDULING RULES
========================================

This is critical:

Only ONE meeting may exist for a given event, date, and time slot.

That means:
- if a meeting is booked for FRC2026 at 10:00 AM in Booth
- then 10:00 AM is unavailable for all other locations in that event

Locations are defined by the Event record.
Meeting time blocks are defined by the Event record.

When creating a meeting:
- Admin must choose an event first
- Then only show sponsors assigned to that event
- Then only show attendees assigned to that event
- Then only allow date/time choices that fit that event’s meeting blocks
- Then only allow locations defined for that event

If a conflicting meeting exists at the same event/date/time:
- prevent save
- show clear validation error
