Redesign Event Scheduling Page Layout to Match Welcome Page

The event scheduling page should follow the same structural layout as the Welcome Page so users experience a consistent interface.

Current problems:

Hero section is oversized

Topics section is missing visual emphasis

Sponsor section spacing is inconsistent

Layout hierarchy differs from the welcome page

We will rebuild the page using the same component layout as the Welcome Page.

1. Page Layout Structure

Replace the current page structure with the following order:

Event Header
Topic Selection Section
Sponsor List Section

This must mirror the Welcome page.

2. Event Header (Match Welcome Page)

Reduce the header height and simplify the design.

Structure:

[Event Logo]

Meeting Scheduling

Browse Sponsors and Schedule Meetings for
The 2026 U.S. BankTech Summit

April 2 – April 4, 2026 • Austin, TX
Event Website

Rules:

Reduce vertical padding

Use the same card style used on the Welcome page

Logo size should match the Welcome page logo

3. Remove the Action Explanation Block

Delete this section entirely:

Available actions include:
• Schedule an onsite meeting during the conference
• Request an online meeting
• Ask a sponsor to send you more information

Reason:

This creates unnecessary visual noise and does not exist on the welcome page.

Styling requirements:

identical topic pill design

same spacing
