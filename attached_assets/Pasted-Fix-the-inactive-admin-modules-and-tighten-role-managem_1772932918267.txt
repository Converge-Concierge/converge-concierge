==================================================

Do NOT make roles fully custom/arbitrary in V1.

- Admin
- Manager
- Sponsor

Each role should have a clear definition in the Users module.

Add a "Role Definitions" panel or help text explaining:

Admin:
- full platform access
- can manage users
- can reactivate archived records
- can delete events/sponsors where allowed
- can edit settings and branding

Manager:
- can manage events, sponsors, attendees, meetings, and reports
- cannot create/edit/delete users
- cannot reactivate archived records
- cannot delete events/sponsors if delete is admin-only
- can edit settings/branding only if explicitly allowed by settings

Sponsor:
- cannot access admin
- can only access their sponsor dashboard via sponsor access links or sponsor access system
- can only see their own event-specific data

In Users:
- keep role selection as a controlled dropdown using only the 3 fixed roles
- do not allow arbitrary custom role creation
- do not add a “create custom role” feature
