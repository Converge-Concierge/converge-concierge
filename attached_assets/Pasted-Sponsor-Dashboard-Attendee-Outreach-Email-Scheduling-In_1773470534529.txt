Sponsor Dashboard, Attendee Outreach, Email Scheduling, Info Requests, Sponsor Health, Reporting, and Admin Navigation

We need to implement the next major set of Concierge improvements. These changes should make the platform more operationally useful for Converge Events, more valuable to sponsors, and stronger as a sponsorship sales tool.

This phase includes updates across:

Sponsor Dashboard

Attendees

Email Center

Info Requests

Sponsor Management

Reporting / Sponsor Health

Automatic Sponsor Reports

Configuration / Admin Navigation

The work should be implemented in a structured way so the UI, workflows, permissions, and automation remain consistent.

Primary Goal

Improve Concierge so it better supports:

sponsor deliverable execution

sponsor engagement monitoring

attendee outreach

email scheduling and email control

sponsor performance analysis

internal sponsorship sales intelligence

1. Sponsor Dashboard — Better File Management / Media Library

Sponsors should be able to manage files more cleanly.

Required behavior

Create a cleaner sponsor-facing media/file management experience for assets such as:

company logos

documents

promo assets

supporting files

Preferred implementation

Add a sponsor-facing Media Library or Files & Assets area within the Sponsor Dashboard.

This can be:

a dedicated tab
or

a clearly structured section within Deliverables

Recommended name:

Files & Assets

or

Media Library
It should support:

upload file

replace file

view file

download file

show file type

show upload date

show which deliverable the file relates to

Do not make sponsors guess where files belong.

10. Attendees — Individual Meeting Scheduling Email

We need the ability to email attendees individually and route them to a meeting scheduling experience.

Required behavior

From the Attendees area, add an action to send an attendee a personalized scheduling email.

Recommended action name:

Send Scheduling Email

This should:

use attendee email

generate secure personalized link

route attendee to a meeting scheduling page

optionally reference sponsor/event context if applicable

This should work without requiring a traditional attendee login.

This phase is complete when:

sponsor reps show registration status

VIP/Registration deliverables check existing registrations

marketing and compliance deliverables are split correctly

attendee list delivery can be scheduled

COI instruction files can be uploaded

Email Center has a Scheduled Emails tab

admins can edit/delete Info Requests

Sponsor Health exists and shows actionable KPIs

benchmark metrics and past performance library exist

sponsor reports can be auto-generated and emailed

sponsor emails are centrally managed

Configuration navigation is cleaned up

Sponsorship Templates is renamed to Sponsorship Packages

25. Success Standard

After this phase, Concierge should feel like a much more mature sponsor operations and sponsor performance platform, giving Converge Events stronger internal controls, better sponsor visibility, better reporting, and better sales intelligence.

That is the standard this phase should meet.
